1️⃣ What is the difference between var, let, and const?
2️⃣ What is the spread operator (...)?
3️⃣ What is the difference between map(), filter(), and forEach()?
4️⃣ What is an arrow function?
5️⃣ What are template literals?


<!-- The Answers is Below -->

1. 
Var: It is a old and traditional way. Using Var, We can face different problems in JS. It can be re-declared and function scoped.

Let: If we use let, we can change variable. It is block scoped, can be updated but not re-declared.

Const: It is most commonly usable. It is block scoped, cannot be updated or re-declared.

2. 
Spread operator: The spread operator is used to expand or copy elements of an array or properties of an object into another array or object.

3. 
Map(): A method that creates a new array by transforming each element of an existing array.

Filter(): A method that creates a new array with elements that pass a given condition.

ForEach(): A method that executes a function for each array element without returning a new array. It is one kind of loop. We can find the properties and values easily by using this loop. 

4. 
Arrow Function: An arrow function is a shorter syntax for writing JavaScript functions. 
like: => 

5. 
Template string (Backtick): Template literals are string literals that use backticks (``) and allow embedding variables and expressions. 
like: ${}
